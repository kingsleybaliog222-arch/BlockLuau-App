# 🟦 BlockLuau App

**Visual Roblox Luau Block Editor** for Android

Like the BlockLuau plugin in Roblox Studio but in your browser/phone.

## Quick Start
1. Open `index.html` in Chrome
2. Drag & drop blocks
3. Tap **Copy Code** → paste in Roblox Studio

## Make it an APK (Easiest)
1. Go to [webintoapp.com](https://webintoapp.com)
2. Paste this repo URL or the index.html content
3. Generate & download APK
